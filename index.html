Installation en 4 étapes · 5 minutes
Fichier MQL4 · Compatible MT4 toutes versions
1
Copie le fichier dans MT4
Dans MT4 → Fichier → Ouvrir le dossier de données → va dans le dossier MQL4/Experts/ et colle GoldTerminalEA.mq4
2
Autorise les requêtes web
Dans MT4 → Outils → Options → Expert Advisors
✓ Coche "Autoriser les requêtes WebRequest"
Ajoute cette URL :
gold-terminal-silk.vercel.app
3
Compile et attache l'EA
Dans MT4 → Éditeur MetaEditor → ouvre GoldTerminalEA.mq4 → clique Compiler (F7)

Ensuite glisse l'EA sur le graphique XAUUSD H1 → coche "Autoriser le trading en direct"
4
Configure les paramètres
Double-clique sur l'EA dans le graphique pour régler :
• Lots = 0 (utilise les lots du signal) ou un montant fixe
• MinConfiance = 65 (%) — seuil minimum
• CheckEvery = 30 (secondes entre chaque vérification)
✅ UNE FOIS INSTALLÉ
Le robot vérifie notre signal toutes les 30 secondes. Quand un signal LONG ou SHORT apparaît avec ≥65% de confiance, il exécute automatiquement le trade dans ton MT4 avec le bon SL et TP.
